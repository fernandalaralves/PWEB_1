#  Programação Web I

Repositório destinado ao armazenamento das atividades e projetos desenvolvidos na disciplina **Programação Web I**.

Curso: Análise e Desenvolvimento de Sistemas  
Instituição: IFCE – Campus Boa Viagem  
Disciplina: Programação Web I  

---

##  Sobre a Disciplina

A disciplina de Programação Web I tem como objetivo introduzir os fundamentos do desenvolvimento web utilizando **HTML**, **CSS** e **PHP procedural**. Durante as unidades, são trabalhados conceitos essenciais como formulários web, estruturas de controle, sessões, cookies e criação de páginas dinâmicas.

##  Estrutura do Repositório

O repositório está organizado por unidades, contendo as atividades desenvolvidas ao longo do semestre.
